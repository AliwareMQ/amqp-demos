# 阿里云 amqp node 客户端 Demo

## 安装

> 如果没有安装 amqplib 请执行如下命令安装
```bash
npm install amqplib --save
```

## 使用方法
消息发送：
```js
node send.js
```

消息接收：
```js
node consume.js
```

## Api 使用
[请参考 amqplib 文档](http://www.squaremobius.net/amqp.node/channel_api.html)

[amqplib 项目地址](https://github.com/squaremo/amqp.node)
